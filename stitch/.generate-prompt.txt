# Foto Galerisi — Ürün Gereksinimleri Dokümanı (PRD)

## 1. Genel Bakış

Foto Galerisi, kullanıcıların fotoğraflarını kategori bazında görüntüleyebildiği, grid layout ile sergileyebildiği ve lightbox ile tam ekran izleyebildiği modern bir web uygulamasıdır. Türkçe arayüze sahip bu uygulama, 20 demo fotoğraf ve kategori filtreleme özelliği sunar. Hedef kitle: fotoğraf meraklıları, portfolyo sahipleri ve görsel içerik seven kullanıcılardır.

## 2. Hedefler

- Kullanıcıya görsel açıdan zengin bir fotoğraf galerisi deneyimi sunmak
- Kategori bazında hızlı ve sezgisel filtreleme sağlamak
- Lightbox ile fotoğrafların tam ekran detaylı görüntülenmesini sağlamak
- Mobil uyumlu ve duyarlı tasarım ile tüm cihazlarda tutarlı deneyim
- Hızlı yükleme ve akıcı geçişler ile performanslı çalışma
- WCAG 2.1 AA erişilebilirlik standartlarına uyum

## 3. Tech Stack Detayı

- **Framework:** React 18.3.x
- **Build:** Vite 6.x
- **Styling:** Tailwind CSS 3.x
- **State:** useState + useReducer (yerleşik React state management)
- **Storage:** localStorage (tema tercihi, filtre durumu)
- **Routing:** React Router DOM 6.x
- **Icons:** Lucide React
- **Image Handling:** HTML5 img tag + CSS object-fit

## 4. Fonksiyonel Gereksinimler

### 4.1 Fotoğraf Grid Görünümü
- Grid layout: responsive, 2-4 sütun arası
- Her fotoğraf: thumbnail görünüm, aspect-ratio korunur
- Hover efekti: scale(1.02) büyüme + gölge artışı
- Lazy loading ile performans
- Fotoğraf altında kategori etiketi görünür

### 4.2 Lightbox Görüntüleme
- Tam ekran modal overlay (siyah arka plan, %90 opaklık)
- Fotoğraf: max-width 90vw, max-height 90vh, object-fit contain
- Kapatma: ESC tuşu, arka plan tıklama, X butonu
- Navigasyon: sol/sağ ok butonları, klavyeden ok tuşları
- Fotoğraf yüklenirken spinner gösterimi

### 4.3 Kategori Filtreleme
- Filtre butonları: horizontal scroll, chips/strip style
- Kategoriler: "Tümü", "Manzara", "Portre", "Şehir", "Doğa"
- Aktif filtre: vurgulu renk (primary), diğerleri nötr
- Filtreleme: anında görsel güncelleme, animasyonlu geçiş
- URL parametresi ile filtreleme durumu kalıcı (opisyonel)

### 4.4 Demo Fotoğraflar
- 20 adet demo fotoğraf (Unsplash API veya placeholder)
- Her fotoğraf: id, url, thumbnailUrl, title, category
- Kategori dağılımı: Dengeli (her kategoriden 4 fotoğraf)
- Fotoğraf boyutları: thumbnail 400x300, full 1920x1080

## 5. Veri Modeli

### Photo Entity
```
{
  id: string
  url: string
  thumbnailUrl: string
  title: string
  category: "landscape" | "portrait" | "city" | "nature"
  width: number
  height: number
}
```

### Category Entity
```
{
  id: string
  label: string (Türkçe)
  value: string
  color: string (hex)
}
```

### AppState
```
{
  photos: Photo[]
  activeFilter: string
  lightbox: { isOpen: boolean, currentPhoto: Photo | null }
  theme: "light" | "dark"
}
```

## 7. Non-Functional Requirements

### 7.1 Performans
- İlk yükleme: < 2 saniye
- Grid render: < 500ms
- Lightbox açılış: < 200ms
- Bundle size hedefi: < 200KB (gzipped)

### 7.2 Erişilebilirlik (WCAG 2.1 AA)
- Klavye navigasyonu: Tab ile tüm elementlere erişim
- Screen reader: ARIA labels, role="button", aria-label
- Kontrast oranı: >= 4.5:1 (text), 3:1 (large text)
- Focus state: visible outline (2px solid primary, offset 2px)
- Touch hedefleri: >= 44x44 px
- ESC ile lightbox kapatma

### 7.3 Tarayıcı Desteği
- Chrome 90+, Firefox 88+, Safari 14+, Edge 90+
- iOS Safari 14+, Android Chrome 90+
- Responsive: mobile 320px → desktop 1920px
